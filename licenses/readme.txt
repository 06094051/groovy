This directory contains generated LICENSE files and snippets used to generate those files.
See the assemble.gradle file for details on how this is done.
Snippets have predefined suffix values in their name to determine which files they go into.
LICENSE (the one for source) and LICENSE-DOC get snippets containing SRC and DOC respectively.
LICENSE-JARJAR gets SRC and JARJAR snippets.
LICENSE-ALLJARJAR gets SRC, JARJAR and ALLJARJAR snippets.
LICENSE-BINZIP gets SRC, JARJAR, ALLJARJAR and BINZIP snippets.
